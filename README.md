# 🍪 Lucky Cookie Bite – Tasksnap Mini Game
一款使用 HTML + CSS + JavaScript 製作的輕量互動小遊戲。  
玩家點擊幸運餅乾，即可獲得隨機祝福、任務或驚喜訊息，為 Tasksnap 增加趣味互動元素。

---

## 🎯 遊戲玩法 How to Play
1. 點擊畫面中的給我好運  
2. 觸發隨機幸運訊息或祝福  
3. 再點擊即可重新抽取  
4. 遊戲支援手機端操作

---

## ✨ 遊戲特色 Features
- 動態餅乾機台動畫  
- 隨機運勢系統  
- 音效回饋與小特效  
- 輕量 HTML 遊戲，可嵌入任何網站  
- 可作為展場互動、品牌延伸內容
